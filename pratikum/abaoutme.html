Assalamualaikum

nama saya Afifah Zulva dan saat ini berusia 21 tahun. saya berasal dari kota padang. saya merupakan mahasiswi Teknik Informatika dari Universitas Riau, saat ini sedang menjalani perkuliahan semester 4. saya merupakan seseorang yang sangat tertarik dengan dunia fotografi, desain grafis dan beberapa aplikasi informasi berbasis web. oleh sebab itu, saya bisa berada di jurusan ini atas dasar ketertarikan.

Beberapa alasan saya memilih Jurusan Teknik Informatika :

  1. pertama, saya tertarik dengan desain kreatif yang tercipta dari teknologi dan saya ingin mempelajari hal tersebut lebih detail lagi.

  3. Ketiga, profesi di bidang IT bisa bekerja dimana pun dan menghemat biaya yang seharusnya dikeluarkan.
  4. keempat, saya mendapat dukungan dari orangtua untuk berkuliah pada jurusan teknik informatika ini.

Hal yang diharapkan setalah lulus tentunya bisa bekerja pada bidang yang diinginkan, saya tertarik pada beberapa bidang pekerjaan yang sesuai dengan jurusan teknik informatika, diantaranya:

  • Software developer
  • System Analyst
  • Web Developer
  • Information Security Analyst
  • Dan masih banyak lagi
